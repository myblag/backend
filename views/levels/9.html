Level 9

I have a very long list of email ids of all my employees but I don’t have a database with all of their names. {{user}}! I need your help! If only there was a fast way to remove ‘@aeproject.co’ from every email ID. Oh wait, there is! Something that I learnt back during my Introduction to CS class. It’s called slicing. Okay, imagine a pre-cut loaf of bread, the kind you get from the supermarket. Now imagine a letter written on each slice. The whole loaf together will form a word or a sentence, or dare I say it, a String! Just like that we can also count the number of slices from 0 onwards Programmers are strange, they count from 0 and not from 1 like 0,1,2,3… . Just like the figure shown below, we can also assign numbers backwards.

Use the following examples in the terminal to get a better idea about how they work.

Say, I want to get the letter `e` from the string, this is what I’m going to do:

x="Hello"
x[1]

Now, that we know how to remove a letter, this is how we remove a substring (a part of a string). Let’s try to take out `ll`.

x="Hello"
x[2:4]

You might be wondering why we used 2:4 instead of 2:3. That’s because the first index number is inclusive (includes that position) and the last index number is exclusive (doesn’t include that position). In other words, the sliced string only includes characters up to the character before the ending index.

Now you’d ask how do we get rid of all letters after a point? It’s simple. Let’s say we need to get all letters after `e` in the string, this is how we’re going to do it:

x="Hello"
x[1:]

And if we only want letters before `o`, we do the following:

x="Hello"
x[:-1]

P.s. keep this in mind, it will be handy for the next level.
