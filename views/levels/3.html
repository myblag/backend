Programming, although a powerful tool, can be a bit weird. There is a difference between 4 and 4.0, can you tell me what it is? Both have the same values but the computer looks at both at differently: 4 has the type called integer (int) and 4.0 has the type float. It’s called a float because it has a decimal point next to it. If you’re ever confused about the type of a variable or an expression you can always use a function in Python called type(variable). Whatever you put in it will return the type of it, be it a single number or a mathematical expression or even a variable.

>>> type(x)   
<class 'int'>

type() is a function which tells you about the datatype of the variable or expression.

>>> x = 60.0  
<class 'float'>
