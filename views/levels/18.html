Lets work with For loops in this level. For loop has a different syntax to be followed, it works with something called as iterator.

Before diving into an example let me introduce you with a function called range.

This function can take upto 3 parameters.

range(x,y,z)

It returns numbers that start from x and goes on till y excluding y and skips z numbers.

For instance, range(0,10,3) would give 0,3,6,9

If only single parameter is passed it return numbers from 0 to that number excluding the said parameter. For example, range(5) would give 0,1,2,3,4

Taking an example of for loop now.

>>> for x in range(5):
...     print(x)
...     
0
1
2
3
4
>>> for x in range(0,50,5):
...     print(x)
...

5

15
20

35
40
45

Here x is the iterator.

Note: for loops do not require any update statements.

def sumList(listSalaries):
    sum = 0

        sum = sum + listSalaries[x]

for loops also provides a more convenient way to solve such problems of iterating through a list of items.

def sumList(listSalaries):
    sum = 0
    for x in listSalaries:
        sum = sum + x 
    return sum

The iterator is assigned values from the list directly.
