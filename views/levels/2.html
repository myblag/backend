Just like in mathematics, programming can too have variables. This might seem new to you but you definity have used it, a few examples being x = 10, y = 20, x + y = 30, etc. Let's look at the few examples given below to get a better understanding.

>>> iHaveAName = 25
>>> x = 60

In the first line of the code shown above, the variable named “iHaveAName” is being set to 25. In the second line, the variable named “x” is set to 60. This is known as an assignment statement, which assigns a value to a variable.

Giving names to things makes it easier to reference them later. It’s much easier to refer to you as “{{user}}” than “That person helping Ada build her Analytical Engine.”

Now that we have names for a few values let’s start calling them by their names.

>>> iHaveAName + x
85
